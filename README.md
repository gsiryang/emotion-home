# 情绪归处

一个移动端优先、可安装、离线可用的本地化网页应用，用于：

- 从身体感受开始识别自身情绪；
- 区分事实、脑补与他人情绪的归属；
- 在缺乏安全感时，以渐进方式练习表达观点、感受、请求与边界；
- 回顾触发模式，而不把情绪记录变成新的考核。

## 本地运行

项目不依赖第三方包。请通过本地 HTTP 服务运行，例如：

```powershell
python -m http.server 4173
```

然后打开 `http://127.0.0.1:4173/`。

## GitHub Pages

将仓库发布源设为仓库根目录即可。所有数据都保存在浏览器本地，不会写入 GitHub 仓库。

## 隐私

- 无账号、无后端、无遥测；
- 默认使用浏览器 `localStorage` 保存记录；
- 可在设置中导出或清除；
- 这是自我觉察工具，不提供医学或心理诊断。
